# momentum

A platformer where speed is your best tool and your biggest risk. The faster and harder you move, the more heat you build up. Push too hard and you overheat - so the real game isn't dodging enemies, it's managing your own momentum through a molten forge world.

**How far can you go before you burn out?**

### Controls:
- **A** / **←** = Left
- **D** / **→** = Right
- **W** / **Space** = Jump

Built with vanilla JavaScript + HTML5 Canvas. No frameworks.
