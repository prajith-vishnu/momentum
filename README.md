# momentum

A game about a machine tearing through a molten forge that never stops heating it up. The longer you run, the hotter you get, and the moment you overheat you get thrown back to your last cooldown. Frost chips are scattered along the way to bring your heat back down, so the real game is chaining from one to the next before the meter fills. Do you play it safe, or push ahead and risk the melt?

**Keep moving or burn out.**

### Controls:
- **W** = Jump
- **A** = Left
- **D** = Right
- **ENTER** = Start / Retry
- **ESC** = Pause
