# momentum

A game about a machine that lives and dies by its own speed. The faster you go the more momentum you build, and momentum is everything, it makes you quicker but it also makes you burn hotter. Stop for even a second and you lose it, so you are always pushing forward, climbing and dropping through a molten forge, grabbing frost chips to cool back down and hopping the spikes waiting to end your run. How far can you get before you melt?

**Build momentum or burn out.**

try it here: https://momentum-phi-ashy.vercel.app

### Controls:
- **W** = Jump
- **A** = Left
- **D** = Right
- **SPACE** = Start / Retry
- **ESC** = Pause
- **M** = Mute
